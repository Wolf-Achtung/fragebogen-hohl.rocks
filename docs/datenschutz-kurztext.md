# Kurztext Datenschutz für die Formularseite

Die Angaben aus diesem Fragebogen werden ausschließlich zur Auswertung der Copilot-Integrationsvoraussetzungen und zur Erstellung einer Implementierungsstrategie verarbeitet. Bitte tragen Sie keine Passwörter, API-Schlüssel, Zugangsdaten oder vertraulichen Kundengeheimnisse ein. Die Übermittlung erfolgt an fragebogen@hohl.rocks. Standardmäßig werden die Angaben nicht dauerhaft auf dem Webserver gespeichert.
